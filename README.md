# federated-mtl
Code used in the experiments of the Master's thesis "Federated Multi-task Learning over Networked Data"

The experiments are written as a jupyter notebook, and it uses PyTorch deep learning library. Also torchvision is required for it contains the MNIST and FashionMNIST datasets.
All libraries used are listed in the requirements.txt file. Note that Anaconda was used for virtual environment control, and the file may contain a lot of additional libraries.
